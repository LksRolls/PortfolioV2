Quelques projets fait en cours

Le restaurant d'entreprise

Le projet avait pour but de développer un site internet en HTML, CSS et PHP pour un restaurant d'entreprise. Le site devait permettre aux employés de consulter les tarifs de celui-ci en fonction de leurs statuts (employé, externe etc.). Ce site a donc une base de données qui permet de stocker les informations sur les employés, les tarifs, les droits etc... Il y a aussi une partie d'administration qui permet de voir, créer, modifier ou supprimer les informations des differentes tables comme un utilisateur, les tarifs ou les prestations

Jeu Pierre Feuille Ciseaux

Le mini projet avait pour but de développer un jeu de Pierre Feuille Ciseaux en HTML, CSS et JS.
Le but du jeu est de jouer contre l'ordinateur. L'utilisateur doit choisir entre Pierre, Feuille ou Ciseaux et l'ordinateur choisit aléatoirement entre les trois choix. Il y a aussi un système de score qui permet de savoir qui a gagné le plus de manche entre l'ordinateur et l'utilisateur.
